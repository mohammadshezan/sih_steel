Place your welcome page reference images here.

Recommended filename: ref.png
Alternative names are fine (e.g., welcome.png). In the app overlay, paste the URL:
  /reference/ref.png  (or the filename you used)

How to use overlay in dev:
1) Run the frontend dev server.
2) Click the "Overlay" button at bottom-right.
3) Paste /reference/ref.png as Image URL.
4) Adjust opacity and scale to align the UI to your image.
